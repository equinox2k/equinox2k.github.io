Class SvgUseElement

The ‘use’ element references another element and indicates that the graphical contents of that element is included/drawn at that given point in the document. The ‘use’ element has optional attributes ‘x’, ‘y’, ‘width’ and ‘height’ which are used to map the graphical contents of the referenced element onto a rectangular region within the current coordinate system.
