Remarks

Many SVG graphics operations use 2×3 matrices. When you need a matrix for matrix arithmetic, you can expand a 2×3 matrix into a 3×3 matrix equivalent by adding a third row of [0 0 1].

CreateRotate(Single, Nullable<Single>, Nullable<Single>)

Creates a rotation matrix about a given point.

If optional parameters and are not supplied the operation corresponds to the matrix [cos(a) sin(a) -sin(a) cos(a) 0 0].

If optional parameters and , the rotate is about the point(x, y) and the operation represents the equivalent of the following specification: translate(x, y) rotate(angle) translate(-x, -y).
