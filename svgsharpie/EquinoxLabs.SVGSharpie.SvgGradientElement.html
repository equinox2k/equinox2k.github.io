Class SvgGradientElement

The SVGGradientElement interface is a base interface used by SvgLinearGradientElement and SvgRadialGradientElement. A gradient element is one that defines a gradient paint server. SVG 1.1 defines the following gradient elements: SvgLinearGradientElement and SvgRadialGradientElement.

GradientTransform

Gets an optional additional transformation from the gradient coordinate system onto the target coordinate system (i.e., userSpaceOnUse or objectBoundingBox).

Declaration
public SvgTransformList GradientTransform { get; }
Property Value
Type Description
SvgTransformList
Remarks

This allows for things such as skewing the gradient. This additional transformation matrix is post-multiplied to (i.e., inserted to the right of) any previously defined transformations, including the implicit transformation necessary to convert from object bounding box units to user space.
